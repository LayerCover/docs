Get LayerCover running in your React dApp in three steps.

---

## 1. Install

```bash
npm install @layercover/sdk viem @mui/material @emotion/react @emotion/styled
```

---

## 2. Add the CoverButton

```tsx
import { CoverButton } from '@layercover/sdk/react';

function ProtocolPage() {
  // Get your viem WalletClient from wagmi
  const { data: walletClient } = useWalletClient();

  return (
    <CoverButton
      walletClient={walletClient}
      poolId={1}
    />
  );
}
```

That's it! The button renders a styled "Cover" button. When clicked, it opens a complete purchase modal with quote fetching, duration selection, and transaction execution.

### Live Example

```tsx
// App.tsx - Full working example
import { CoverButton } from '@layercover/sdk/react';
import { useWalletClient } from 'wagmi';

export default function App() {
  const { data: walletClient } = useWalletClient();

  return (
    <div style={{ padding: '2rem', textAlign: 'center' }}>
      <h2>My DeFi Protocol</h2>
      <p>Your deposits are protected by LayerCover</p>
      <CoverButton
        walletClient={walletClient}
        poolId={1}
        buttonText="Get Protected"
        onSuccess={() => alert('Coverage purchased!')}
      />
    </div>
  );
}
```

## Full CoverButton Props

| Prop | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `walletClient` | `WalletClient` | ‚úÖ | viem WalletClient instance |
| `poolId` | `number` | ‚úÖ | Pool ID for the coverage pool |
| `availableBalance` | `number` | | User's token balance (display only) |
| `onSuccess` | `() => void` | | Callback after successful purchase |
| `referralCode` | `string` | | Your referral code (earn premiums from referrals) |
| `buttonText` | `string` | | Button label (default: "Cover") |
| `theme` | `Theme` | | Custom theme overrides |

### Example with All Options
