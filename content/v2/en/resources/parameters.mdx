## How Parameters Change

All pool parameter updates follow a transparent, on-chain governance process:

| Actor | Capability | Safeguard |
| :--- | :--- | :--- |
| **Protocol Owner** | Configure pool parameters, premium shares, risk limits | Changes via `ProtocolConfigurator` with on-chain transparency |
| **Committee** | Emergency pool pause, incident reporting | Cannot modify core configuration |
| **UMA Oracle** | Resolve disputed claims via Optimistic Oracle | Decentralised, permissionless dispute process |
