LayerCover core smart contracts are released under the **Business Source License 1.1 (BUSL-1.1)**. The code is fully auditable, but commercial use or forking is restricted until the specific Change Date defined in each repository.

All other assets including frontends, SDKs, and off-chain infrastructure remain **closed source**.
