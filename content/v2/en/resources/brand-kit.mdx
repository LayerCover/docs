## Typography

### Primary Font: Inter

**Usage:** Body text, UI elements, general content

**Weights:**
- Regular (400) - Body text
- Medium (500) - Subheadings, emphasis
- Semibold (600) - Headings
- Bold (700) - Strong emphasis

**Download:** [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)

**CDN:**
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

**CSS:**
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

### Monospace Font

**Usage:** Code blocks, technical content, contract addresses, technical identifiers

**Font:** System monospace stack

**CSS:**
```css
font-family: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace;
```

This provides optimal code rendering across different platforms without requiring font downloads.
